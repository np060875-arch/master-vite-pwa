MASTER - MLBB Chat Game (Vite + React + PWA)
========================================

Hướng dẫn nhanh:

1) Giải nén zip:
   cd master-vite-pwa

2) Cài dependencies:
   npm install

3) Chạy dev:
   npm run dev
   -> Mở http://localhost:5173 trên máy hoặc điện thoại trong cùng mạng

4) Build để deploy:
   npm run build
   -> Thư mục dist/ sẽ chứa build để upload (Netlify / Vercel)

Lưu ý: rosters.json gồm tên đội mẫu được lấy từ nguồn công khai (MDL, MPL, VMC). Bạn có thể chỉnh sửa 'src/data/rosters.json' để cập nhật tên cầu thủ thật.
